Section 2: ID Number

The ID number shall contain 3 sets of digits. The first set being the State where the person's primary home is located. The second set of digits is a counter for how many people have joined the world. The third set of digits is a 5 digit number that is completely randomized and is unique to that person in that particular State.
        If the person's primary home is not in a State or district that has a number, then they will be given the number 0 for the first set of digits.
